REQUIRED ICON FILES
===================

Place the following icon files in this folder before running electron-builder:

  icon.ico    — Windows icon (256x256, ICO format)
  icon.icns   — macOS icon (1024x1024, ICNS format)
  icon.png    — Linux icon (512x512, PNG format)

Use a tool like https://www.icoconverter.com/ to convert PNG → ICO.

Suggested design:
- Background: #2563EB (blue)
- Symbol: White document/invoice icon
- Size: 512x512 or higher source PNG
